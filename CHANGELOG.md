# CHANGELOG
## Release 2.10 - Feb 27, 2026
- Добавлены кнопки редактирования названия колоды, дублирования колоды, очистки и удаления.
- Исправлена ширина баннера описания карты

## Release 2.9 - Feb 22, 2026
- Добавлена база знаний с правилами игры

## Release 2.8 - Feb 20, 2026
- Добавлены экспериментальные функции по отображению описаний карт:
 -- макрос /i меняет шрифт на курсив
 -- макрос //0 позволяет сделать принудительный перенос на следующую строку
 -- длина вмещаемого текста в строку увеличина

- Обновлен лаунчер, теперь он может обновлять сам себя


## Release 2.7 - Dec 16, 2025
- Добавлена возможность создания карты Оружия для редактора
- Оптимизирован процесс генерации карт, в связи с этим:
-- Карты генерируются на ходу при введении любого изменения
-- Картинку в редакторе можно протаскивать и наблюдать изменения на ходу
- Новодобавленные карты теперь автоматически экспортируются в папку с игрой

## Release 2.6 - Dec 11, 2025
- Исправлено отображение карт в библиотеке без фильтра - больше нет большого пустого пространства.
- При переключении фильтров в библиотеке позиция скролла сохраняется пропорционально предыдущей позиции. 
- Токены временно больше не добавляются в сайдборд при создании колод, в связи с новой системой активаций карт.
- Добавлены более удобные уведомления вместо всплывающих окон.
- При загрузке колоды из базы стандартные карты распознаются корректно.  
- Сетевые фиксы
